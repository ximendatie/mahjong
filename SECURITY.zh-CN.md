# 安全政策

mahjong 会观察本地 Agent 活动，因此必须谨慎处理用户数据。请报告任何可能超出文档说明而暴露私密 prompt、session 元数据、本地路径或 provider 文件的问题。

## 支持版本

项目目前尚未到 1.0。正式 release 分支出现之前，安全修复会优先进入默认分支。

## 报告漏洞

请不要在公开 issue 中包含私有 session 文件或完整对话内容。你可以提交一个最小化 issue，描述风险、受影响 provider，以及不暴露隐私的复现步骤。如果之后增加私密联系渠道，本文件会同步更新。

## 数据处理原则

- 本地观察应保持只读。
- 不应修改 provider 的配置、session 或缓存文件。
- 默认不展示完整对话正文。
- 不应在缺少明确文档、用户控制和 review 的情况下新增远程网络调用。
